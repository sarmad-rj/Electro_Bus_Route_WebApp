# Project Overview

This is a Spring Boot project named "Electro Bus Route". It appears to be a web application given the inclusion of the Thymeleaf dependency. The project is built using Apache Maven and Java 17.

# Building and Running

## Building the project

To build the project, run the following command in the terminal:

```bash
mvn clean install
```

## Running the application

To run the application, use the following command:

```bash
mvn spring-boot:run
```

The application will start on the default port (usually 8080).

## Running tests

To run the tests for the project, use the following command:

```bash
mvn test
```

# Development Conventions

*   The project follows the standard Maven project structure.
*   Source code is located in `src/main/java`.
*   Resources, such as configuration files, are in `src/main/resources`.
*   The main application class is `SoftCoders.ElectroBusRoute.ElectroBusRouteApplication`.
